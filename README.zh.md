## 两种开始方式

### Mac mini 版 — 主力产品

预配置的 Mac mini 邮寄到家。AI 在本地运行，数据在本地。云端提供 LLM 网关、记忆同步、数据备份——就像你的 AI 的 iCloud。

- 你的 AI 住在你桌上的盒子里，不是硅谷的服务器
- 断网也能用（优雅降级）
- 一个 U 盘就能带走一切
- 适合：对隐私有刚需的专业人士（律师、医生、高管）

### 云端版 — 先试试

无需硬件，浏览器内完整体验。30 天免费。

- 体验"被 AI 记住"是什么感觉
- 试用期积累的记忆和人格，购买 Mac mini 后无缝迁移
- 转化驱动不是功能限制——是记忆留存

两个版本使用体验完全一致：飞书或微信发一条消息，AI 立即响应。

## 工作原理

```
  你的设备                              云端服务
  ────────                              ────────

  [Mac mini / 云端容器]
   AI Agent 运行时  ──── API ────→     LLM 网关       （智能模型路由）
   本地数据          ──── 同步 ──→     记忆服务       （结构化、可检索）
   完整 Home 目录   ──── 备份 ──→     备份服务       （增量同步、一键恢复）
```

无论 Agent 跑在哪里，都消费同一套云服务。你不关心 AI 住在哪里——你关心它懂不懂你。
