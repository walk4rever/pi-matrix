# pi-matrix

**你的专属数字员工，开箱即用。**

[English](./README.md)

pi-matrix 是一款智能助手产品，通过飞书或微信与你的数字员工对话。无需任何技术背景，无需配置，无需运维。

---

## 两种开始方式

### Mac mini 版
预配置好的 Mac mini 直接邮寄到你手中。数字员工完全运行在你自己的硬件上，数据不离开你的本地环境。

### 云端版
无需任何硬件。我们为你托管一个独立的云端实例，开通账号，几分钟内即可开始对话。

两个版本的使用体验完全一致：在飞书或微信里发一条消息，数字员工立即响应。

---

## 为什么选择 pi-matrix

**用户零负担**
不需要安装任何软件，不需要命令行，不需要做任何决定。数字员工始终在线，随时待命。

**真正的隐私保护**
你的数据归你所有。云端数据可随时导出或删除。Mac mini 版数据完全保留在本地。

**始终在线**
设备与实例自动运行，故障自动恢复。我们全天候监控运行状态，你无需操心。

**持续进化**
能力升级通过云端远程交付，你的数字员工会持续变得更智能，无需你做任何操作。
