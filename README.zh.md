# pi-matrix

**你的专属数字员工，开箱即用。**

[English](./README.md)

pi-matrix 是一款软硬件一体化的智能助手产品。我们将先进的 AI Agent 能力预装在 Mac mini 上，通过飞书或微信直接与你的数字员工对话，无需任何技术背景，插电联网即可使用。

---

## 为什么选择 pi-matrix

**真正的隐私保护**
你的数据归你所有。所有本地计算在你的设备上完成，云端数据完全由你掌控，可随时导出或删除。

**零门槛上手**
不需要安装任何软件，不需要命令行，不需要配置。设备开机后，在飞书或微信里发一条消息，你的数字员工就在线了。

**始终在线**
设备自动运行，断电重启后自动恢复。云端实时监控设备状态，确保你的数字员工随时待命。

**持续进化**
我们通过云端远程更新能力持续不断地提升数字员工的智能水平，你无需做任何操作。

---

## 产品形态

- **硬件**：预配置 Mac mini，插电联网即用
- **交互**：飞书 / 微信，使用你已经熟悉的工具
- **云端**：配置、记忆、数据云端同步，多设备一致体验
- **安全**：端到端数据隔离，用户数据互不可见
